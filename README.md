## 📌 개발 목표
* 코로나 감소세로 방역 조치가 단계적으로 완화되며 관광 산업이 다시 활성화되기 시작
* 이에 여행사에서 이용할 수 있는 직원 관리 시스템 개발을 목표로 프로젝트 진행
* 직관적인 UI와 가벼운 기능을 포함한 간단한 직원 관리 프로그램
